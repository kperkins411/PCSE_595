    Week Lectures/Readings Examples
    1 Part1: Automate setup of a cloud based GPU instance
    Cloud based machines tend to start with a minimal configuration. To be useful they need extensive changes that are tedious and time consuming to set up (aliases, installing software, configuration files, creating debugging environments etc). One problem is that once the machine shuts down, all of these changes are lost, which means the whole process is repeated the next time the its started. This unit covers creating a startup script to automate this process.
    The lectures are practical in that I will walk thru this process explaining every step.
